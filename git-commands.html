Git Commands

  • git --version
  • git config --global user.name userName
  • git config --global user.email userEmail@domain.com
  • git config user.name
  • cd pathToUrl/folderName
  • cd ..
  • pwd ( Present Working Directory )
  • ls
  • ls -la
  • mkdir folderName
  • touch fileName ( create file )
  • rm fileName ( remove file )
  • rmdir folderName
  • rm -rf .git ( if you want to delete whole .git folder )
  • git init
  • git status
  • git add fileName
  • git add . ( for staging all )
  • git add --a ( for staging all )
  • git rm --cached fileName
  • git rm fileName.extension
  • git mv fileName.extension fileNameRename.extension
  • git commit -m "commit message here"
  • git commit -a -m "Direct Commit" ( It commit all the changes without staging it )
  • git log ( see all commits )
  • git log --oneline ( see commits in one line )
  • git log -p ( tell every changes which was done )
  • git log -p -2 ( number of commits show )
  • git log --stat ( in short show all logs )
  • git log --pretty=oneline ( it shows all logs in one line )
  • git log --pretty=short
  • git log --since=2.days
  • git log --since=2.week
  • git log --since=2.months
  • git log --since=2.years
  • git log --pretty=format:"%h -- %an"
  • git log --pretty=format:"%h -- %ae"
  • git restore --staged fileName.extension
  • git diff file.extension
  • git diff
  • git diff --staged file.extension
  • git reset fileName.extension ( unstaged the files )
  • git checkout fileName.extension ( remove recent changes from file )
  • git checkout -- fileName.extension ( remove recent changes from file )
  • git checkout . ( remove recent changes from all files )
  • git checkout -f ( if you want to remove all changes and move to previous commit )
  • git checkout ID ( to move to the following ID base commit )
  • git checkout master ( move to master commit )
  • git revert ID ( shift+ : -> to get out with screen )
  • git reset ID ( to get to the following commit without deletion of files )
  • git reset ID --hard ( to get to the following commit and delete the files )
  • git branch branchName ( create branch )
  • git branch -a ( look for all branches )
  • git branch -v ( show branchName, hash and last commit )
  • git checkout branchName ( switch to other branch )
  • git branch -D branchName ( delete the branch )
  • git branch --merged ( already merged branches )
  • git branch --unmerged ( not already merged branches )
  • git checkout -b branchName ( create the branch and switch to branch )
  • git push origin --delete branchName ( delete branch from remote system )
  • git merge branchName
  • git push https://github.com/userName/project-xyz.git master
  • git remote add origin https://github.com/userName/project-xyz.git
  • git clone https://github.com/userName/zs-portfolio.git
  • git remote -v
  • git push origin main/BranchName
  • git push -d origin BranchName
  • git pull
  • git pull origin main/BranchName
  • git config --global alias.st status ( If you want to create custom commands )
