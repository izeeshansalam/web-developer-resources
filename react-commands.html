React Commands

  • node -v ( to check the node version installed on your machine )
  • cls ( clear the command prompt )
  • npx create-react-app appName ( to create new react app )
  • cd folderName ( to change the directory )
  • cd.. ( move to back directory )
  • code . ( to open directory in vs code )
  • npm run start ( to start the react app )
  • npm install ( install node modules )
  • npx json-server --watch data/db.json --port 8000 ( to create fake rest api )
  • npm install react-router-dom@5 ( to create react router dom version 5 )
  • node app/FileNameWithoutExtension ( fire the file code using node )
  • npm init ( to initialize node and create package.json file )
  • npm install express -save ( install express dependencies package of node )
  • npm uninstall express ( to uninstall express package )
  • npm install -g nodemon
  • nodemon app.js
  • npm install ejs -save
  • npm install body-parser
  • npm install mongoose -save
